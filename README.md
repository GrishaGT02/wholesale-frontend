# Frontend - Агрегатор оптовых поставщиков

React приложение на TypeScript для управления заказами.

## Технологии

- **React 19** + **TypeScript**
- **Vite** - сборщик
- **Tailwind CSS** - стилизация
- **React Router** - роутинг
- **TanStack Query** - управление серверным состоянием
- **Axios** - HTTP клиент
- **date-fns** - работа с датами

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно на http://localhost:5173

## Сборка

```bash
npm run build
```

## Структура проекта

```
src/
├── components/      # Переиспользуемые компоненты
│   ├── OrdersTable.tsx  # Таблица заказов
│   └── Tabs.tsx         # Компонент табов
├── pages/          # Страницы приложения
│   ├── Dashboard.tsx   # Главная страница с заказами
│   └── Login.tsx        # Страница входа
├── services/        # API клиент
│   └── api.ts          # Настройка Axios и API методы
├── types/           # TypeScript типы
│   └── index.ts        # Типы для API
├── lib/             # Утилиты
│   └── utils.ts        # Вспомогательные функции
└── App.tsx          # Главный компонент с роутингом
```

## Переменные окружения

Создайте файл `.env`:

```env
VITE_API_URL=http://localhost:8000
```
